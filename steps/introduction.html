Introduction

Impress.js è un framework per creare presentazioni liberamente ispirato a Prezi

rispetto al suo ispiratore, impress.js ha alcuni vantaggi (imho)

  • è un progetto open source e gratuito
  • offre possibilità illimitate (a patto di essere abbastanza creativi)
  • è basato su CSS3
Presenta principalmente uno svantaggio: se non siete pratici di HTML e CSS può diventare ostico creare una presentazione (attualmente non esistono editor WYSIWYG)
